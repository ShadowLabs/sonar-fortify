ABSTRACT

The SOAP Monitor module allows attackers to sniff SOAP traffic.

EXPLANATION

Apache Axis 2 provides developers with a utility to monitor incoming and outgoing SOAP messages through a Java applet. The SOAP Monitor will show all SOAP messages used to invoke a Web Service. Attackers can use the utility to eavesdrop on traffic between a Web Service and its clients.
