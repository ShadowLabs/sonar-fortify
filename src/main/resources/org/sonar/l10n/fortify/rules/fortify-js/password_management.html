ABSTRACT

Storing a password in plaintext can result in a system compromise.

EXPLANATION

Password management issues occur when a password is stored in plaintext in an application.


In this case, the password is read into the program at in at line and used to access a resource at in at line .



Example: The following code uses a hardcoded password to connect to an application and retrieve addressbook entries: 


    ...

    obj = new XMLHttpRequest();

    obj.open('GET','/fetchusers.jsp?id='+form.id.value,'true','scott','tiger');

    ...


This code will run successfully, but anyone who accesses the containing web page will be able to view the password.
